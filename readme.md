\# Day 18 Inventory Management System



\##  Overview

This project is a basic inventory management system developed using only:

\- Arrays  

\- Loops  

\- Conditional statements  



No functions, structures, or pointers are used as per assignment constraints.



The program stores up to 10 products and allows operations such as displaying all items, searching by ID, and calculating inventory value.



\## Features

\- Add up to 10 products  

\- Display all products  

\- Calculate total inventory value  

\- Find highest valued product  

\- Find lowest valued product  

\- Search product by ID  

\- Menu-driven design
